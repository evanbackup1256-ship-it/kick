Quick Start

Get running in 3 steps

Copy the loader, inject with your executor, and pick a game from the hub.

  1. 1

    Copy the loader

    Use the script above or tap Copy Script — it auto-updates from GitHub.

  2. 2

    Execute in Roblox

    Open your executor, paste the loadstring, and run it in any supported game. Not sure yours works? Check live WEAO status.

  3. 3

    Pick a game

    Check live status below — green means working. Join the game and the hub menu loads automatically.
